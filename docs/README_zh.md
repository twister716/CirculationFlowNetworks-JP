# Circulation Flow Networks（环流网络）

---

## 概述

受够了在机器之间铺设成堆的线缆，还要为每一台新设备重新规划走线？

环流网络用节点取代线缆：放下节点，范围内的发电机、用电器、储能设备自动连入网络，能量即刻流通。网络天然是无线的——节点之间互相感知、自动链接，覆盖范围随你扩建而生长，中继节点帮助你更好的让它延伸到任何你需要的地方。

放下中枢，网络就有了管理中心——插件系统、频道隔离、权限控制，全部搞定。

走进充能节点的范围，身上的装备自动补充能量。

给中枢装上频道插件，多个网络接入同一频道即可共享能量——哪怕它们分布在不同维度。

**不只是方便：**
- **能量体系兼容** — 原生支持 FE，并提供 API 供模组开发者注册自定义能量兼容类型
- **频道与权限** — 支持公共、团队、私有三种模式，内置 FTB Teams 支持
- **为大型网络而生** — 针对大规模网络优化，不拖累服务器性能
- **开发者友好** — 完整 API，第三方模组可扩展节点类型与能量处理器

是时候告别线缆时代，开启全新的能源传输体验了。

---

## 快速上手

1. 合成一个**节点基座**和一个**端口节点**
2. 放置节点基座，将端口节点放在基座上方
3. 在端口节点的能量范围（默认 8 格）内放置发电机和用电器
4. 网络自动检测机器并开始传输能量——无需任何额外操作

**扩展网络：**
- 多放几组基座 + 端口节点，节点之间在链接范围内会自动连接
- 用**中继节点**桥接距离较远的区域
- 放下**中枢**解锁插件、频道和权限管理

**提示：**
- 使用**环流配置器**潜行右键节点可查看网络连接状态和范围可视化
- 如果机器的类型识别有误（如发电机被识别为用电器），切换到**能源节点配置模式**，潜行右键该机器即可手动覆写为正确的类型

## 方块

### 节点基座

节点的放置基础。端口节点、充能节点和中继节点必须放置在基座上方。破坏基座会使上方的节点掉落。

### 端口节点

能量网络的基本组成单元。自动检测范围内的能源机器并将它们纳入网络。大多数情况下，你只需要这种节点就能搭建一个完整的能量网络。

- 能量检测范围：8 格
- 链接范围：12 格

### 充能节点

为范围内玩家背包中的物品无线充电。需要通过中枢的充能配置界面指定充能的装备栏位。适合放置在工作区或常驻基地入口。

- 充能范围：5 格
- 链接范围：8 格

### 中继节点

不检测能源机器，专门用于延伸网络的覆盖距离。当两组节点之间距离过远无法直连时，在中间放置中继节点即可桥接。

- 链接范围：20 格

### 中枢

网络的可选管理中心。提供 GUI 界面，解锁插件系统、频道系统和权限管理。每个网络最多一个中枢。中枢本身也能检测能源机器和为玩家充能。

- 能量检测范围：10 格
- 充能范围：8 格
- 链接范围：16 格
- 插件槽位：5 个

### 环流屏蔽仪

在指定范围内阻止能量流动，可用于隔离网络区段或在特定条件下切断供电。支持红石控制：

- **常规模式**：收到红石信号时阻止能量流动
- **反转模式**：收到红石信号时允许能量流动

## 物品

### 环流配置器

多功能检查与配置工具，具有两个主模式：

**网络信息模式** — 潜行 + 右键节点

- 显示全部信息
- 显示连接范围
- 显示节点网络

**能源节点配置模式** — 潜行 + 右键能源机器，手动覆写类型检测

- 能量输出（发电机）
- 能量输入（用电器）
- 能量存储
- 清除配置

操作：潜行 + 右键空气切换主模式，潜行 + 滚轮切换子模式。

### 中枢频道插件

插入中枢后，将该网络接入指定频道。接入同一频道的多个网络可以共享能量资源，实现跨网络甚至跨维度的能量互通。频道信息存储在物品上，转移插件即转移频道绑定。

### 广域充能插件

插入中枢后，网络的充能范围扩展到整个维度——同维度内所有具有权限的玩家都能享受无线充能，无论距离多远。

### 维度充能插件

插入中枢后，充能范围突破维度限制——所有具有权限的玩家无论身处哪个维度都能接收充能。

## 中枢界面

中枢提供多个功能标签页：

| 标签   | 功能                       |
|------|--------------------------|
| 节点列表 | 查看网络中所有节点，双击可定位到世界坐标     |
| 充能配置 | 选择哪些玩家接收充能，按装备栏位分类控制     |
| 插件列表 | 管理中枢的 5 个插件槽位            |
| 频道列表 | 浏览和切换可用频道                |
| 权限设置 | 管理频道成员角色（所有者 / 管理员 / 成员） |
| 频道设置 | 创建与删除频道                  |
| 能源显示 | 实时查看网络能源状态               |

## 频道与权限

网络通过**频道**隔离。每个频道有三种可见性模式：

| 模式     | 行为                                 |
|--------|------------------------------------|
| **公开** | 所有玩家都能看到并作为成员使用该频道                 |
| **团队** | 仅与频道所有者同队的玩家可见并可用（内置 FTB Teams 支持） |
| **私有** | 仅频道所有者和被显式授权的玩家可见并可用               |

每个频道具有独立的权限系统：

- **所有者** — 频道创建者，拥有全部权限，不可转移
- **管理员** — 可管理网络配置和成员
- **成员** — 可使用网络

显式授权的权限优先于可见性模式的默认判定。例如，即使频道设为私有，被显式添加为成员的玩家仍可使用。

## 网络连接机制

每个节点具有**链接范围**属性，处于彼此链接范围内的节点自动建立连接，组成网状拓扑。链接类型包括双向连接和单向连接。

能源检测节点（端口节点、充能节点、中枢）会在自身的**能源范围**内扫描能源机器，并自动识别机器类型（发电机 / 用电器 / 储能设备）。识别有误时可通过环流配置器手动覆盖。

同一网络不允许存在多个中枢，放置时会检测冲突。

## 配置

所有参数均可在配置文件中调整：

### 节点范围

| 节点    | 参数     | 默认值 |
|-------|--------|-----|
| 端口节点  | 能量检测范围 | 8   |
| 端口节点  | 链接范围   | 12  |
| 充能节点  | 充能范围   | 5   |
| 充能节点  | 链接范围   | 8   |
| 中继节点  | 链接范围   | 20  |
| 中枢    | 能量检测范围 | 10  |
| 中枢    | 充能范围   | 8   |
| 中枢    | 链接范围   | 16  |
| 环流屏蔽仪 | 最大范围   | 8   |

### 其他选项

| 配置项      | 默认值 | 说明                          |
|----------|-----|-----------------------------|
| 默认能源单位显示 | FE  | 界面中默认显示的能源单位                |
| 动态模型渲染   | 开启  | 中继节点与节点基座的动画效果，关闭可提升性能      |
| 能源实体黑名单  | —   | 按类名前缀排除不兼容的方块实体，使其不被识别为能源容器 |
| 供应操作黑名单  | —   | 排除特定设备的通用能源供应，仅允许专用节点连接     |
